# Conversational AI ChatBot

An intelligent chatbot built with Microsoft's DialoGPT transformer for natural, human-like conversations. [DialoGPT](https://huggingface.co/microsoft/DialoGPT-medium) is a pretrained model for multi-turn dialogue, designed to generate responses comparable to human quality.

## Files
- `Conversational_AI_ChatBot.py`: Python script for the chatbot.
- `conversational-ai-chatbot.ipynb`: Interactive notebook version.
- `requirements.txt`: Python library dependencies.

## ChatBot Types
- **Rule-based**: Responds according to pre-defined rules. No learning or adaptation.
- **Retrieval-based**: Selects responses by retrieving relevant information from a document.
- **Intelligent**: Uses machine learning to understand and generate responses, simulating human-like conversation.
